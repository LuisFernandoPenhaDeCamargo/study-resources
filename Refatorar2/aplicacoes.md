# Aplicações

### Ubuntu

`Ctrl+Alt+Seta_direcional_para_esquerda/direita`**:** muda de ambiente de trabalho (Desktop Environment). Seta para a esquerda, muda para a esquerda. Seta para a direita, muda para a direita, além de criar um novo ambiente de trabalho, mas isso somente se o atual possuir alguma aplicação aberta.

### Tilix

- `CTRL + ALT + A`**:** abre uma nova aba (um split)
- `ALT + NUMERO`**:** muda para a aba do número especificado
- `CTRL + SHIFT + T`**:** abre um novo terminal
- `CTRL + Page Up ou Page Down`**:** muda de terminal (`Up` diminui a numeração, `Down` aumenta. Os comandos dão a volta nos terminais)
- `CTRL + SHIFT + Page Up ou Page Down`**:** altera o identificador númerico do terminal

### VS Code

`CTRL + SHIFT + Pipe`**:** se estiver na abertura do símbolo, vai para o seu fechamento e vice e versa. Exemplo: estou na abertura de uma chave (`{`), vou para o seu fechamento (`}`), se estivesse no seu fechamento, iria para a sua abertura. Isso funciona para chaves, parênteses e colchetes.

### DBeaver
