1)

Sobre a criação de um documento HTML e suas tags, podemos afirmar:

I. Pode ser utilizado um editor de textos simples, como o bloco de notas, ou mesmo editores mais avançados, como o Notepad++ e o Visual Studio Code.

II. Um documento HTML tem como extensão .html e deve sempre ser salvo com o nome de “index”.

III. Para salvar um arquivo como um documento HTML é necessário garantir que ele seja salvo com o tipo e, consequentemente, a extensão “.html”.

IV. O uso de IDEs ou editores mais robustos de código permitem utilizar certas facilidades, como o destaque de códigos e palavras reservadas das linguagens de programação ou marcação.

V. Um desenvolver web que cria documentos HTML pode ser chamado também de programador em HTML. 

São verdadeiras:

Alternativas:

Resolução comentada:

A afirmação I é verdadeira, pois pode-se utilizar qualquer editor de textos para criar um documento HTML. Assim como a afirmação III também é verdadeira, pois, para salvar um arquivo como documento HTML é necessário garantir o uso da extensão “.html”, sendo indicado tal tipo no momento de salvar o arquivo correspondente. Finalmente, também é verdade o que se afirma em IV, pois os editores de código ou IDEs mais robustos trazem facilidades, como a identificação de palavras e códigos reservados de linguagens de programação ou de marcação, como o HTML. A afirmação II é falsa, uma vez que um documento HTML pode receber outro nome, além de “index”, contudo, para que o navegador consiga inicializar um projeto de uma página, ao menos um arquivo deve ser nomeado assim, em um projeto. A afirmação V também é falsa, pois como HTML é uma linguagem de marcação, torna-se errado dizer que uma pessoa é um programador HTML. O termo programador deve ser utilizado somente no contexto de linguagens de programação.

Alternativas:

Resolução comentada:

O seletor de classe e de identificador são os que necessitam de modificação nos elementos HTML, por meio da indicação de um atributo de nome class e id, respectivamente. O seletor universal e de grupo são diferentes. O primeiro aplicará as regras do CSS para todos os elementos do documento HTML, já o de grupo, só aplicará as regras para os elementos indicados no seletor. Não existe um seletor de atributos. 9)

Um código JavaScript para ser interpretado e, consequentemente, realizar suas funções em uma página HTML, necessita ser integrado ao código de tal página. Considerando o excerto de código JavaScript a seguir, assinale a alternativa correta.

document.getElementById("conteudo").innerHTML

Alternativas:

Resolução comentada:

A palavra document faz referência ao código do documento HTML, contudo, este será modificado por meio do código JavaScript. Para tal modificação será necessário buscar o elemento HTML por meio de seu id, o que é feito pelo método getElementById. Logo, não se trata de identificadores de folhas de estilos. O conteúdo a ser mostrado é passado então para o innterHTML, que corresponde ao conteúdo interno do elemento HTML alvo. O trecho de código não precisa ser inserido em um evento HTML, pois, neste caso, o próprio script identifica o elemento HTML pelo seu id. Finalmente, a palavra “conteudo” é o valor dado ao id do elemento HTML. Caso não exista um elemento com tal identificador, o código JavaScript não será executado.
