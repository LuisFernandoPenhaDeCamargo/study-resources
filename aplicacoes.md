# Tilix

### Comandos

- `CTRL + ALT + A`**:** abre uma nova aba;
- `ALT + NUMERO`**:** muda para a aba do número especificado;
- `CTRL + SHIFT + T`**:** abre um novo terminal;
- `CTRL + PAGE UP ou PAGE DOWN`**:** muda de terminal (`UP` diminui a numeração, `DOWN` aumenta. Os comandos dão a volta nos terminais);
- `CTRL + SHIFT + PAGE UP ou DOWN`**:** altera o identificador númerico do terminal.

# VS Code

### Comandos

- `CTRL + SHIFT + PIPE`**:** se estiver na abertura do símbolo, vai para o seu fechamento e vice e versa. Exemplo: estou na abertura de uma chave`{`, vou para o seu fechamento `}`; se estivesse no seu fechamento, iria para a sua abertura.\
    Isso funciona para chaves, parênteses e colchetes.
